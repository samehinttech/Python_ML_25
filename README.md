## Setup Instructions
1. Clone the repository
    ```bash
    git clone https://github.com/samehinttech/Python_ML25.git
   ```
2. Navigate to the project directory:
    ```bash
    cd Python_ML25
    ```
3. Create a virtual environment:
   ```bash
   python -m venv .venv
   ```
4. Activate the virtual environment:
      ```bash
      .venv\Scripts\activate
      ```
5. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```
6. Start Jupyter Notebook (for interactive notebooks) and avoid files path issues:
    ```bash
   jupyter notebook
    ```
   > **Important:** Always start Jupyter from the project root directory (`Python_ML_25`) to ensure correct file paths.
7. Run Python scripts:
    ```bash
   python script_name.py
    ```
   
Happy coding!
